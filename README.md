NodeJs Project

* Chief editor can:
    *Create News
    *Fetch News
    *Update News
    *Delete News
    *Search News

* login route
    * check if user exists
    * compare user's password with stored has
    * create a token
    * send token to user
    * authenticate book route (GET)

    * Seeding
